React Native采坑(持续更新)

发表于 2018-10-14 更新于 2023-06-23 分类于 React Native

环境搭建时报某某文件找不到

解决：尝试升级你的 cli

VSCode报错 ‘type aliases’ can only be used in a .ts file.ts(8008)

解决：打开 VSCode 扩展，输入 @buildin 筛选选择 TypeScript and JavaScript Language Features 这款插件 disable 掉即可。

某些原生库如果没有链接进来，会报 undefined is not an object(evaluating ‘xxx’) 错误

解决：react-native link xxx 或手动链接进来

main.jsbundle 丢失

解决：删掉之前绑定，跑一下 react-native bundle --entry-file index.js --platform ios --dev false --bundle-output ios/main.jsbundle --assets-dest ios 重新生成，需要留意 AppDelegate.m 中的相关设置

‘config.h’ file not found

解决：React Native 日常报错 ‘config.h’ file not found

error: Build input file cannot be found: libfishhook.a

解决：问题-react-native无法运行

远程调试出现无限循环的”callbacks_poll”，这个问题是有realm框架发出的

解决：Infinite loop of “callbacks_poll” ，完整路径：node_modules/realm/lib/browser/rpc.js

‘xxx is not a function’

解决：可能是this没有绑定该方法

JavaScript浮点数计算出现精度丢失

解决：可以使用mathjs库

react native如何全局导入模块，类似iOS的pch文件中import

解决：react native中有个全局对象模块global（其实是Node.js的），为global添加属性就能全局使用，使用的时候不需要global前缀。

打包时出现/node_modules/react-native-root-siblings/index.js: /node_modules/react-native-root-siblings/index.js: Exporting local “_default”, which is not declared. (This is an error on an internal node. Probably an internal error.)的问题。

解决：进入到node_modules/react-native-root-siblings/index.js中，修改export default class { … }为export default class ClassName { .. }，ClassName随便命个名。详见
