条件查询

  • 时间筛选：YYYY-MM-DD@HH:MM:SS:NANOSECONDS和YYYY-MM-DDTHH:MM:SS:NANOSECONDS，NANOSECONDS可省略
  • 比较运算符：=/ ==，<=，<，>=，>，和!=/ <>(不知道这是什么符号)，支持int，float，double，和Date，但bool只支持一部分运算符
  • string，data(ArrayBuffer)支持=（和==）， !=（和<>），BEGINSWITH，CONTAINS和ENDSWITH，例如name CONTAINS 'Ja'；string还支持LIKE，模糊匹配。
  • [c]不区分大小写，需要用在CONTAINS，例：CONTAINS[c] 'Ja'。
  • 连接运算符：AND/ &&，OR/ ||和NOT/ !。
  • 列表属性运算符：@count，@size，@min，@max，@sum，@avg(平均值)，例：company.employees.@count > 5，检索出员工数量多于5个的公司。
  • $0，$1…用于表示变量，类似 C 语言的%f，例："cars.@sum.miles > $0", average，$0代表average

监听

Realm，Results和List对象都支持addListener监听。下面是两种不同类型的通知

  • Realm Notifications：提交写入事务时通知的简单回调。
  • Collection Notifications：在插入，删除和更新时接收更改元数据的回调。
